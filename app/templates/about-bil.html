About Bitcoin Is Like

Data

The most important part of any data project is, of course, the data. Bitcoin Is Like uses two data sources:

Quand, a fantastic resource for any social or economic data series you can think of. They have free and premium data sets, as well as a great API and client libraries for absolutely any language or software you might find yourself using. Seriously, Quandl. We use their Google Finance data, for now.

BitcoinAverage.com, a bitcoin price aggregator. It's an open source project that collects price data from exchanges across the web in at least a dozen currencies. Their API gives you access to most of the data, as well as the historical data that Bitcoin Is Like relies on.
