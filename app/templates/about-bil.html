About Bitcoin Is Like

Data

The most important part of any data project is, of course, the data. Bitcoin Is Like uses two data sources:

Quand, a fantastic resource for any social or economic data series you can think of. They have free and premium data sets, as well as a great API and client libraries for absolutely any language or software you might find yourself using. Seriously, Quandl. We use their Google Finance data, for now.

BitcoinAverage.com, a bitcoin price aggregator. It's an open source project that collects price data from exchanges across the web in at least a dozen currencies. Their API gives you access to most of the data, as well as the historical data that Bitcoin Is Like relies on.

Technology

Bitcoin Is Like wouldn't have been possible without a lot of great, open source technology. Specifically:

  • Python
  • Flask

  • Pandas
  • JavaScript
  • jQuery

  • PostgreSQL

And many, many more

This project started as a rejected hackathon idea. Not that we implemented it and it was rejected - we went with another idea (a location-based meetup app that I still say did not get the respect it deserved). The idea was that some people buy and hold bitcoin as a store of value - and that's crazy. If you bought at a certain point in 2014, it would have been as if you'd bought and held the Ukranian Hyrvnia - something you probably wouldn't do as a sound investment strategy. So I wanted to see what else bitcoin was like. Thus...Bitcoin Is Like
